    3. Ease of Use

    4. Debugging and Error Handling
        - ChatGPT: Users can debug by by pasting error codes and messages into the chat, but they need to frequently whitch between the code and the chat tab. In addition, the suggested solutions are not always completely accurate.
        - Cursor: Debugging is faster and more efficient since changes can be made and tested directly in the editor, and the suggested fixes tend to be more accurate.

    5. Flexibility and Customization
        - ChatGPT: Adding major new features can be relatively complex, but small customizations (such as changing the game speed) are easy because they only require minor edits at the top of the code.
        - Cursor: The code is more flexible because its sturcture is well organized. Adding new features and customization are easier.
